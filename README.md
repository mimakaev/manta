# manta-hic
Deep learning network &amp; utilities for predicting Hi-C data from sequence

# Conventions 

## Tensor sizes 

Throughout the project we use the following convention for tensor sizes: 

* B - batch size 
* C - number of channels
* N - number of bins 
* H - number of heads
